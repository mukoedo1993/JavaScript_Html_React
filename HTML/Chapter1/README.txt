Targets:
	1: Elements: See how HTML describes the structure of a web page
	2: Learn how tags or elements are added to your document
	3: Write your first web page

Summary:
	1:HTML pages are text documents.
	2: HTML uses tages(characters that sit inside angled brackets)
	to give the information they surround special meaning.
	3: Tags usu. come in pairs. The opening tag denotes the start of a 
	piece of content; the closing tag denotes the end.
	4: Opening tags can carry attributes, which tell us more about the 
	content of that element.
	5: Attributes require a name and a value.
