There are several types of information 
that need to be displayed in a grid or table.
For example: sports results, stock reports, 
train timetables.

When representing information in a table, you need to think in terms of a grid made up of rows
and columns(a bit like a spreadsheet). In this chapter you will learn how to:
1: Use the four key elements for creating tables
2: Represent complex data using tables
3: Add captions to tables
