1: The Document Object Model(DOM)
The DOM is a structured representation of The
UI of a client-side web application that is, at least
initially, built from the HTML code of a web application
that is, at least initially, built from the HTML code of
a web application. 

2: Events
A huge majority of Javascript applications are event-driven
applications, meaning that most of code is executed in the context
of a response to a particular event.

e.g.: Network events, timers, and user-generated events 


3: Browser API:
The browser offers an API that allows us to access infor
about devices, store data locally, or communicate with 
remote servers. We'll explore some of these APIs throughout
